┌─────────────────────────────────────────────────────────────────────────────────────┐
│ EXEMPLOS DE FLUXO                                                                   │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                      │
│  CENÁRIO 1: Novo colaborador regista-se                                             │
│  ──────────────────────────────────────                                             │
│  1. Email adicionado a /whitelist/{email} com registado=false                       │
│  2. Acede ao site, clica "Registar"                                                 │
│  3. Sistema valida email na whitelist e cria:                                       │
│     - Utilizador em Authentication                                                  │
│     - Documento em /employees/{email} com role="colaborador"                        │
│  4. Colaborador consegue aceder a formações e certificados                          │
│                                                                                      │
│  CENÁRIO 2: Gestor de conteúdos cria nova formação                                 │
│  ───────────────────────────────────────────────────                               │
│  1. Acede a Admin > Formações > Criar Nova                                          │
│  2. Preenche: Título, Módulos, Quiz, Data limite                                   │
│  3. Sistema cria /courses/{courseId} + /courses/{courseId}/modules/{mid}            │
│  4. Faz upload de PDFs: /storage/pdfs/{courseId}/{moduleId}.pdf                    │
│  5. Publicar: formação fica visível a todos colaboradores                          │
│                                                                                      │
│  CENÁRIO 3: Gestor de RH promove colaborador a gestor                              │
│  ──────────────────────────────────────────────────────                             │
│  1. Acede a Admin > Utilizadores                                                    │
│  2. Pesquisa colaborador                                                            │
│  3. Edita: role="colaborador" → role="gestor_conteudos"                            │
│  4. Sistema atualiza Firestore                                                      │
│  5. Colaborador vê automaticamente novas opções ao fazer login                      │
│                                                                                      │
│  CENÁRIO 4: Colaborador conclui curso e ganha certificado                          │
│  ─────────────────────────────────────────────────────────                          │
│  1. Completa módulo + passa quiz                                                    │
│  2. Progresso em /employees/{email}/progress/{courseId} ≥ 80%                      │
│  3. Sistema sugere "Emitir certificado" (Gestor conteúdo)                          │
│  4. Gestor clica "Emitir"                                                          │
│  5. Sistema cria /certificados/{email}/{certId}.pdf                                │
│  6. Colaborador pode descarregar na Dashboard                                       │
│                                                                                      │
└─────────────────────────────────────────────────────────────────────────────────────┘
